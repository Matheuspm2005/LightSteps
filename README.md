# 📖 LightSteps

LightSteps is a web application designed to help users follow Bible reading plans and test their knowledge through quizzes.

## 🚀 Features

* 📚 Bible reading plans (Old Testament, New Testament, Full Bible)
* 🧠 Quizzes for each book of the Bible
* 📊 Progress tracking
* 🔐 User authentication (register, login, password reset)

## 🛠️ Technologies Used

* Python (Flask)
* SQLite
* HTML, CSS, JavaScript
* Bootstrap

## ⚙️ Installation

1. Clone the repository:


git clone https://github.com/Matheuspm2005/LightSteps.git
cd LightSteps

2. Create a virtual environment:

python -m venv venv
venv\Scripts\activate

3. Install dependencies:

pip install -r requirements.txt

4. Run the application:

flask run

## 📌 Notes

* The database is not included. You may need to create or seed it manually.
* Environment variables should be stored in a `.env` file.
